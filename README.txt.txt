# final-prodject
* Kad pasileisti jums reikes maven kuri galima rasti cia: https://maven.apache.org/download.cgi , taipat jo irasimo  instrukcijas galima rasti cia: https://maven.apache.org/install.html
* Taip pat reike parsisiusti Intellij kuri galima gauti is cia: https://www.jetbrains.com/idea/download/#section=windows , irasimo instrukcijas galima rasti cia: https://www.jetbrains.com/help/idea/installation-guide.html#silent
* Atidarius programa galima paleisti projekta nusikopijavus repository url, ieskokit mygtuko pavadinimu "Get from VSC" ir i URL laukyli iklijuokite sy url https://github.com/Vytautas-T/final-prodject
* tai pat jum reike susikurti duomenu baze pavadinimu: knygos_rezervacijos_duomenu_baze , naudojant utf8_general_ci abcle.
* paleidus programa pirma syki susikur admin ir user paskyros: login - admin , password - pass . login - user , password - pass . paleidus is naujo daugiau sykiu tai gali dublikuotis to isvengti galima pakeitus `boolean alreadySetup = false;` i `boolean alreadySetup = true;` SetupDataLoader klaseje.
